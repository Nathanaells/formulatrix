# formulatrix

Repository for Formulatrix Bootcamp — C# learning exercises and projects.

## Structure

| Project | Description |
|---------|-------------|
| `ConsoleApp` | Sandbox for experimenting with C# fundamentals |
| `LogicExercise` | Practice problems and logic exercises |

## Getting Started

1. Install [.NET SDK](https://dotnet.microsoft.com/download)
2. Clone the repository
3. Open `formulatrix.slnx` in your IDE or run from the CLI:

```bash
dotnet build
dotnet run --project ConsoleApp
```
